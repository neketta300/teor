## 📖 Пошаговый разбор

### ШАГ 1: Генерация данных

```python
data = pd.DataFrame({
    'year': np.random.randint(2000, 2024, 300),        # Год 2000-2023
    'mileage_km': np.random.randint(10000, 300000, 300), # Пробег
    'engine_volume': np.random.choice([1.4, 1.6, 2.0, 2.5, 3.0], 300),
    'horsepower': np.random.randint(90, 300, 300),
    'fuel_type': np.random.choice(['бензин', 'дизель', 'электро'], 300, p=[0.6, 0.3, 0.1])
})

# Цена = базовая - возраст - пробег + объём + мощность + тип_топлива + шум
data['price'] = (
    50000
    - (2024 - data['year']) * 1000      # Каждые год старения = -1000 руб.
    - data['mileage_km'] * 0.05          # Каждый км = -0.05 руб.
    + data['engine_volume'] * 2000       # Каждый литр объёма = +2000 руб.
    + data['horsepower'] * 100           # Каждая л.с. = +100 руб.
    + np.where(data['fuel_type'] == 'дизель', 2000, 0)    # Дизель +2000
    + np.where(data['fuel_type'] == 'электро', 5000, 0)   # Электро +5000
    + np.random.normal(0, 3000, 300)     # Шум (разброс)
)
data['price'] = data['price'].clip(lower=5000)  # Цена не может быть < 5000
```

**Как работает формула:**
- Новая машина (2024 года, пробег 0) = ~50 000 руб.
- 10-летняя машина = 50 000 - 10×1000 = 40 000 руб.
- С пробегом 100 000 км = ещё -5000 руб.
- Объём 2.0 л = +4000 руб.
- Мощность 150 л.с. = +15 000 руб.

### ШАГ 4: 5 моделей регрессии

#### 1. Linear Regression (Линейная регрессия)

```python
lr = LinearRegression()
lr.fit(X_train_scaled, y_train)
```

**Как работает:**
```
price = w1*year + w2*mileage + w3*volume + w4*hp + w5*diesel + w6*electro + b
```

Находит такие коэффициенты `w1...w6`, чтобы минимизировать сумму квадратов ошибок.

**Плюсы:** Быстрая, интерпретируемая (коэффициенты = вклад каждого признака).
**Минусы:** Не ловит нелинейные зависимости.

---

#### 2. Ridge Regression (Ридж-регрессия)

```python
ridge = Ridge(alpha=1.0)
ridge.fit(X_train_scaled, y_train)
```

**Отличие от линейной:** Добавляет **штраф** за большие коэффициенты.

```
Обычная: минимизирует Σ(y_true - y_pred)²
Ridge:   минимизирует Σ(y_true - y_pred)² + α × Σ(wi²)
```

**Зачем:** Предотвращает **переобучение**. Когда признаки сильно коррелируют, обычная регрессия даёт огромные коэффициенты. Ridge их «прижимает» к нулю.

**Параметр `alpha`:**
- `alpha = 0` → обычная линейная регрессия
- `alpha = 1` → умеренный штраф
- `alpha = 100` → очень сильный штраф (все коэффициенты ≈ 0)

---

#### 3. Lasso Regression

```python
lasso = Lasso(alpha=0.1)
lasso.fit(X_train_scaled, y_train)
```

**Отличие от Ridge:** Может **обнулять** коэффициенты полностью → отбор признаков!

```
Lasso: минимизирует Σ(y_true - y_pred)² + α × Σ|wi|
```

**Пример:** Если `horsepower` неважен, Lasso сделает `w_horsepower = 0`. Ridge сделает его маленьким, но не нулём.

**Когда использовать:** Когда много признаков и нужно отобрать самые важные.

---

#### 4. Random Forest Regressor

```python
rf = RandomForestRegressor(n_estimators=100, random_state=42)
rf.fit(X_train, y_train)  # НЕ нужно масштабирование!
```

**Как работает:**
```
Дерево 1: если year > 2015 и mileage < 50000 → price ≈ 35000
Дерево 2: если volume > 2.0 и horsepower > 150 → price ≈ 40000
...
Дерево 100: если fuel_type == 'электро' → price ≈ 45000

Итог = среднее всех 100 деревьев
```

**Плюсы:** Ловит нелинейности, устойчив к выбросам, не требует масштабирования.
**Минусы:** Медленнее, сложнее интерпретировать.

---

#### 5. Gradient Boosting Regressor

```python
gb = GradientBoostingRegressor(n_estimators=100, random_state=42)
gb.fit(X_train, y_train)
```

**Как работает (последовательное обучение):**
```
Модель 1: предсказывает цену → ошибка = 5000 руб.
Модель 2: учится предсказывать ОШИБКУ модели 1 → остаток = 2000 руб.
Модель 3: учится предсказывать ОШИБКУ модели 2 → остаток = 800 руб.
...
Итог = модель_1 + модель_2 + модель_3 + ...
```

**Отличие от Random Forest:**
- RF: 100 деревьев обучаются **параллельно** (какое независимо)
- GB: 100 деревьев обучаются **последовательно** (какое исправляет ошибки предыдущего)

**Плюсы:** Часто лучшая точность.
**Минусы:** Легко переобучиться, нужен подбор параметров.

---

### Метрики регрессии

| Метрика | Формула | Единицы | Когда использовать |
|---------|---------|---------|-------------------|
| **MAE** | Σ\|y_true - y_pred\| / n | В рублях | Понятная интерпретация: «в среднем ошибка = X руб.» |
| **MSE** | Σ(y_true - y_pred)² / n | В рублях² | Штрафует большие ошибки |
| **RMSE** | √MSE | В рублях | Как MAE, но больше штрафует выбросы |
| **R²** | 1 - SS_res/SS_tot | Безразмерная [−∞, 1] | Доля объяснённой дисперсии |

**Пример:**
```
Реальные цены:    [30000, 40000, 50000]
Предсказания:     [32000, 38000, 47000]
Ошибки:           [2000,  2000,  3000]

MAE  = (2000 + 2000 + 3000) / 3 = 2333 руб.
MSE  = (2000² + 2000² + 3000²) / 3 = 5 666 667
RMSE = √5 666 667 = 2380 руб.
```

**R² подробно:**
- `R² = 1.0` → идеальное предсказание (все точки на линии)
- `R² = 0.9` → модель объяснила 90% разброса цен
- `R² = 0.0` → модель не лучше, чем просто предсказать среднюю цену
- `R² < 0` → модель хуже, чем среднее (переобучение)

### ШАГ 5: Сравнение моделей

```python
models = {
    'Linear Regression': LinearRegression(),
    'Ridge': Ridge(alpha=1.0),
    'Lasso': Lasso(alpha=0.1),
    'Random Forest': RandomForestRegressor(n_estimators=100),
    'Gradient Boosting': GradientBoostingRegressor(n_estimators=100)
}

for name, model in models.items():
    # Линейные модели → масштабированные данные
    if name in ['Linear Regression', 'Ridge', 'Lasso']:
        model.fit(X_train_scaled, y_train)
        y_pred = model.predict(X_test_scaled)
    else:
        model.fit(X_train, y_train)
        y_pred = model.predict(X_test)

    mae = mean_absolute_error(y_test, y_pred)
    rmse = np.sqrt(mean_squared_error(y_test, y_pred))
    r2 = r2_score(y_test, y_pred)

    print(f"{name}: MAE={mae:.0f}, RMSE={rmse:.0f}, R²={r2:.4f}")
```

**Типичный результат:**
```
Linear Regression:  MAE=3200, RMSE=4100, R²=0.8500
Ridge:              MAE=3250, RMSE=4150, R²=0.8470
Lasso:              MAE=3300, RMSE=4200, R²=0.8440
Random Forest:      MAE=2100, RMSE=2800, R²=0.9300  ← лучший
Gradient Boosting:  MAE=1900, RMSE=2500, R²=0.9450  ← ещё лучше
```

**Почему деревья лучше:** Цена зависит нелинейно. Например, пробег 200 000 км влияет на цену иначе, чем пробег 10 000 км. Линейная модель этого не ловит.

### ШАГ 7: Кросс-валидация

```python
from sklearn.model_selection import cross_val_score

rf = RandomForestRegressor(n_estimators=100, random_state=42)
cv_scores = cross_val_score(rf, X, y, cv=5, scoring='r2')

print(f"R² по фолдам: {cv_scores}")
print(f"Средний R²: {cv_scores.mean():.4f}")
print(f"Стандартное отклонение: {cv_scores.std():.4f}")
```

**Зачем:** Один split train/test может быть «удачным» или «неудачным». Кросс-валидация даёт более надёжную оценку.
